## Аннотация

В данном репозитории представлены файлы по проекту по исследованию задачи Упаковки по корзинам. 
Проблема упаковки по корзины — классическая \textbf{NP}-трудная комбинаторная задача. В работе исследуются границы её аппроксимируемости и приводится доказательство того, 
что для любого сколь угодно малого $\varepsilon > 0$ не существует полиномиального алгоритма, строящего $(3/2 - \varepsilon)$-приближенное решение, если \textbf{P} $\neq$ \textbf{NP}. 
С другой стороны, в работе описывается и реализуется алгоритм, который для любого $\varepsilon > 0$ находит решение, использующее не более $(1 + \varepsilon)OPT + 1$ корзин, где $OPT$ — оптимальное число корзин. 

В данном репозитории представлены файлы с реализрцией и тестированием этого алгоритма.

## Описание файлов

### Упаковка_по_корзинам_проект.pdf

Непосредственно текст проекта. Содержит теоритические обоснования и описание результатов. 

### BinPacking.py 

Этот файл содержит реализованный клас **BinPackingAeps**. Он имеет один атрибут -- значение $\varepsilon$. Его основная функция **solve** по набору весов предметов возвращает найденное разбиение предметов на корзины, использующее не более $(1 + \varepsilon)OPT + 1$ корзин, где $OPT$ -- оптимальное количество корзин для ранного набора. 

### lemma_2_1.py

Этот файл посвящен реализации алгоритма, описанного в Лемме 2.1. основной рабоы. Основная функция -- **lemma2_1(items)**. Она получает на вход набор весов предметов, все веса неменьше $\varepsilon$ и всего различных весов не более $K$. И возвращает разбиение по корзинам в виде набора списков индексов предметов в исходном массиве. 

### tests,py

Этот файл содержит 3 функции для генерации тестов различных типов -- для проверки корректности и для исследования времени работы. 

### main.py

В этом файле реализовано тестирование алгоритма. Реализованы функции проверки корректности алгоритма и зависимости времени работы от параметров с помощью генерации нескольких наборов данных с помощью функций из файла **tests.py**.
